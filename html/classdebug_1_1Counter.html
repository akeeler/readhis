Detailed Description

Counter for simple debugging.

Shows message on any action. Main counter is static so you may create many instances across the code. The subindex helps to trace problematic brake points. The optional text may be passed as a stringstream, so it's easy to pass variables etc.

Example: debug::Counter cnt; stringstream ss;

//... some code

ss << "sz = " << sz; cnt.set(ss);

for (int i = 0; i < sz; ++i) { cnt.add(); //... code }
